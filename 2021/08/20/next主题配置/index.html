04 正确姿势打开标签页和分类页

  1. 命令hexo n page tags,source文件夹中多出一个包含index.html的tags文件夹。这时候博客中的“标签就能打开了。
  2. 修改index.html文件。title改为标签，增加一行type: tags（这句不写标签页不会有标签）。
  3. 命令：hexo n page categories创建分类页。
  4. 修改categories里的index.html ，与之前类似。
